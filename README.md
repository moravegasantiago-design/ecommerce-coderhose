# Ecommerce

## Descripción
Ecommerce es una tienda en línea desarrollada con **HTML, CSS y JavaScript**, que permite a los usuarios navegar, buscar y comprar productos de manera intuitiva. Incluye varias funcionalidades avanzadas para mejorar la experiencia del usuario.

---

## Características principales

- **UI moderna y responsiva**: Interfaz limpia y funcional.  
- **Alertas y notificaciones**: Librerías integradas para mostrar mensajes dinámicos al usuario.  
- **Búsqueda y filtros**: Filtrado por tags, nombre y referencia de productos.  
- **Base de datos en Firebase**: Manejo de productos, usuarios y carrito.  
- **Carrito de compras**: Agregar productos, ver detalles y cantidad de items.  
- **Ver producto**: Detalle completo del producto al hacer clic.  

---

## Tecnologías usadas

- **Frontend:** HTML, CSS, JavaScript  
- **Base de datos:** Firebase  
- **Librerías:** Para alertas y notificaciones (por ejemplo, SweetAlert2)
